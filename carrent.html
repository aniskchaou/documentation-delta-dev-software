Megasteel - ERP Business Solution - CodeCanyon Item for Sale Preview Item Screenshots Add to Favorites Add to Collection Megasteel is an open-source integrated management software package comprising a large number of modules to effectively manage all of the information and operational services of a mechanical construction company (ERP). This Web application is delveloped using Spring boot and thymleaf.
